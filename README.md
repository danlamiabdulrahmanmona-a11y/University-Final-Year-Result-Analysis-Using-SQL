# 🎓 University Final Year Result Analysis Using SQL

A comprehensive SQL project that analyzes university final-year student results using MySQL. The project demonstrates database creation, data import, and analytical SQL queries to evaluate student academic performance, graduation status, CGPA distribution, and departmental insights.

---

# 📖 Project Overview

This project was developed to analyze final-year university student records and generate meaningful academic insights using SQL.

The database stores student information such as CGPA, department, programme, graduation status, degree classification, and course information. SQL queries are then used to answer common academic and administrative questions.

---

# 🎯 Project Objectives

- Design and create a relational database.
- Import student academic records into MySQL.
- Analyze student performance.
- Evaluate graduation outcomes.
- Identify top-performing students.
- Generate academic KPIs.
- Support data-driven academic decision-making.
